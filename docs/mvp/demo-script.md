## [0:30 - 1:30] 스타 피처 — 원클릭 배포

**화면**: 원클릭 배포 플로우

**자막/나레이션**:
> "바이브 코딩 입문자를 위한 원클릭 배포.
> 템플릿 선택 → GitHub 연결 → 배포 버튼 하나로 끝."

**액션**:
1. 대시보드 → "원클릭 배포" 메뉴 클릭
2. 6종 템플릿 중 "디지털 명함" 선택
3. (사전에 GitHub 연결 완료 상태) GitHub 계정 확인
4. "배포 시작" 버튼 클릭
5. 배포 진행 상태 표시 (애니메이션 + 타이머 + 팁)
   - 레포 생성 → Pages 설정 → 빌드 → 완료
6. 배포 완료 → 라이브 사이트 링크 클릭
7. 실제 배포된 홈페이지 보여주기
8. "내 사이트" 목록에서 관리 화면 간단히 보여주기

**강조 포인트**:
- "코드 한 줄 없이 3분만에 GitHub Pages 배포"
- "완전 무료 호스팅 (GitHub Pages)"
- "Git 명령어를 전혀 몰라도 됩니다"

## [1:30 - 2:30] 개발자 기능 — 서비스맵, 환경변수, 커맨드 팔레트

**화면**: 프로젝트 대시보드 → 서비스맵 → 환경변수

### [1:30 - 1:50] 대시보드 + 서비스맵

**자막/나레이션**:
> "성장한 개발자에게는 더 강력한 도구를.
> 프로젝트에 연결된 모든 서비스를 한눈에."

**액션**:
1. 프로젝트 대시보드 (Bento 레이아웃) 보여주기
2. 서비스맵 탭 클릭 → React Flow 인터랙티브 다이어그램
3. 노드 드래그, 줌인/아웃, 연결선 확인
4. Frontend / Backend / DevTools 레이어 구분 표시

### [1:50 - 2:15] 환경변수 관리

**자막/나레이션**:
> "API 키와 환경변수를 AES-256 암호화로 안전하게 관리.
> 환경별 분류, .env 다운로드, GitHub Secrets 동기화까지."

**액션**:
1. 환경변수 탭 → 등록된 변수 목록
2. "복호화" 버튼으로 값 확인 (마스킹 → 표시)
3. ".env 다운로드" 버튼 클릭
4. GitHub Secrets 동기화 패널 간단히 보여주기

### [2:15 - 2:30] 커맨드 팔레트

**자막/나레이션**:
> "Ctrl+K로 어디서든 빠르게 이동."

**액션**:
1. Ctrl+K → 커맨드 팔레트 열기
2. 프로젝트 검색 → 바로 이동
3. 테마 전환 (다크/라이트)
4. 언어 전환 (한국어/영어)
