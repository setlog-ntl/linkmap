# Linkmap 포지셔닝 전략

> 분석 기준일: 2026-02-10

## 1. Linkmap 현재 기능 요약

| 영역 | 기능 |
|------|------|
| 서비스 관리 | 23+ 서비스 카탈로그 (Supabase, Stripe, OpenAI 등) |
| 시각화 | React Flow 기반 서비스 맵 |
| 환경변수 | AES-256-GCM 암호화, 멀티환경 (Dev/Staging/Prod) |
| 헬스 체크 | 8개 전용 어댑터 + 제네릭 폴백 |
| 온보딩 | 5단계 설정 마법사, .env 임포트, 스마트 서비스 제안 |
| 팀 협업 | RBAC, 25명 팀 지원, 감사 로그 |
| 결제 | Stripe 연동 (Free/Pro/Team) |
| 플랫폼 | MCP 서버, CLI, API 토큰 |
| i18n | Korean-first, 영어 확장 준비 |

## 2. 시장 포지셔닝 다이어그램

```
                        기능 복잡 (Enterprise)
                              │
                   Vault      │      Infisical
                              │        Cortex
                              │
    전문가 ────────────────────┼──────────────────── 초보자
                              │
               Backstage      │      Doppler
               Configu        │       Port
                              │
                         dotenvx
                              │
                        기능 단순 (개인)


                        ╔═══════════╗
                        ║  Linkmap  ║  ← 초보자 × 적정 기능
                        ╚═══════════╝
                        (우측 중앙 영역)
```

### Linkmap의 위치: "초보자 × 적정 기능"

- **dotenvx**보다 기능 풍부 (웹 UI, 시각화, 팀 기능)
- **Doppler**보다 초보자 친화적 (한국어, 설정 마법사, 더 저렴)
- **Infisical/Vault**보다 훨씬 단순 (Enterprise 기능 의도적 배제)
- **Port**와 다른 축 (서비스 포털이 아닌 "환경변수 + 연결 관리")

## 3. 5가지 핵심 차별점

### 차별점 1: 유일한 "환경변수 + 서비스맵" 결합

> 10개 경쟁사 중 환경변수 관리와 서비스 연결 시각화를 동시에 제공하는 서비스는 **Linkmap만 유일**하다.

- 시크릿 관리 도구 (Infisical, Doppler 등)는 시각화가 없다
- 개발자 포털 (Backstage, Port 등)은 환경변수 관리가 없다
- Linkmap은 이 두 영역의 **교차점**에 존재한다

### 차별점 2: 바이브코딩 초보자 타깃 (시장에 없음)

> 기존 시크릿 관리 도구는 모두 **DevOps 엔지니어** 또는 **숙련 개발자**를 타깃으로 한다.

- Vault: "수주~수개월" 온보딩, CLI 전문 지식 필요
- Infisical: 5단계 리소스 계층, Identity/RBAC/SCIM 용어
- Doppler: 가장 쉽지만 여전히 개발자 전제
- **Linkmap**: "바이브코딩으로 첫 프로젝트를 만드는 사람"이 3분 안에 시작 가능

### 차별점 3: Korean-first UI

> 한국어를 기본 지원하는 환경변수/시크릿 관리 도구는 **시장에 전무**하다.

- 10개 경쟁사 모두 영어 전용 (일부 문서 번역은 있으나 UI는 영어)
- 한국의 바이브코딩 붐 + 한국어 UI = 진입 장벽 최소화

### 차별점 4: 간결함을 핵심 가치로

> 대부분의 경쟁사는 기능을 추가하며 복잡해지는 반면, Linkmap은 **의도적으로 간결함을 유지**한다.

| 경쟁사가 복잡해지는 이유 | Linkmap의 선택 |
|--------------------------|---------------|
| Infisical: PKI, PAM, 동적 시크릿 추가 | 환경변수 관리에 집중 |
| Vault: 암호화 서비스, 시크릿 엔진 확장 | 필요한 것만 AES-256 |
| Backstage: 플러그인 생태계 복잡화 | 23개 서비스 템플릿 내장 |
| Cortex: 스코어카드/메트릭 과부하 | 헬스 체크 = 초록/빨강 |

### 차별점 5: AI 도구 네이티브 통합 (MCP)

> MCP 서버를 내장하여 Claude Code, Cursor 등 AI 코딩 도구와 직접 연동하는 환경변수 관리 도구는 **Linkmap이 유일**하다.

- 바이브코딩 사용자가 AI 도구를 통해 환경변수를 관리할 수 있음
- CLI 없이 AI 채팅으로 "Stripe 키 추가해줘" 가능
- 경쟁사는 모두 CLI 또는 웹 대시보드만 제공

## 4. 벤치마킹 제안 항목

경쟁사의 검증된 기능 중 Linkmap 철학(초보자·간결함)에 맞게 재해석할 수 있는 항목:

### 4-1. Secret Versioning → "되돌리기" 기능
**출처**: Infisical (Pro), Doppler

| Infisical 방식 | Linkmap 적용안 |
|----------------|---------------|
| 모든 시크릿 변경을 버전으로 기록 | 최근 5개 변경만 보관 |
| 버전 간 diff 비교 | "이전 값" / "현재 값" 단순 비교 |
| Point-in-time 복구 | "되돌리기" 원클릭 버튼 |

**구현 난이도**: 중간 | **사용자 가치**: 높음 (실수 복구)

### 4-2. Secret Reference → DATABASE_URL 자동 조합
**출처**: Infisical

| Infisical 방식 | Linkmap 적용안 |
|----------------|---------------|
| `${env.VAR}` 문법으로 시크릿 참조 | 서비스 템플릿에서 자동 URL 생성 |
| 사용자가 레퍼런스 수식 작성 | "Supabase 연결 URL 자동 생성" 버튼 |
| 범용 참조 시스템 | 서비스별 연결 문자열 전용 |

**예시**: Supabase 서비스에서 HOST, PORT, USER, PASSWORD, DB_NAME을 입력하면:
```
DATABASE_URL=postgresql://[user]:[password]@[host]:5432/[db]
```
자동 생성 → 복사 또는 환경변수로 바로 저장

**구현 난이도**: 낮음 | **사용자 가치**: 매우 높음 (초보자가 가장 헤매는 부분)

### 4-3. 노코드 서비스 연결 → 가이드형 서비스 추가
**출처**: Port

| Port 방식 | Linkmap 적용안 |
|-----------|---------------|
| 자동 서비스 발견 (인프라 스캔) | package.json / .env 스캔으로 서비스 자동 감지 |
| 노코드 포털 빌더 | 감지된 서비스 원클릭 추가 |
| 200+ 통합 | 기존 23개 카탈로그 활용 |

**구현 난이도**: 중간 | **사용자 가치**: 높음 (온보딩 단축)

### 4-4. 스코어카드 → 프로젝트 건강도 점수
**출처**: Cortex

| Cortex 방식 | Linkmap 적용안 |
|-------------|---------------|
| DORA 메트릭, 운영 준비도 등 복합 스코어 | 단순 100점 만점 건강도 |
| 수십 개 기준 항목 | 5개 기준만 사용 |
| 엔터프라이즈 대시보드 | 서비스 맵 위에 점수 표시 |

**5가지 건강도 기준**:
1. 모든 필수 환경변수 설정됨 (20점)
2. 서비스 헬스 체크 통과 (20점)
3. 프로덕션 환경 분리됨 (20점)
4. 최근 7일 내 환경변수 갱신 (20점)
5. 팀 멤버 접근 권한 설정됨 (20점)

**구현 난이도**: 낮음 | **사용자 가치**: 중간 (동기부여 게이미피케이션)

### 4-5. EnvKey 종료 → 이탈 사용자 흡수
**출처**: EnvKey 서비스 종료 (2025.02.01)

| 기회 | 실행안 |
|------|--------|
| EnvKey 사용자가 대안을 찾고 있음 | "EnvKey에서 이전하기" 가이드 작성 |
| EnvKey의 핵심 가치: "간편한 환경변수 관리" | Linkmap의 핵심 가치와 동일 |
| 오픈소스 CLI 도구만 남음 | 웹 UI + 시각화로 차별화 |

**구현 난이도**: 매우 낮음 (마케팅 콘텐츠) | **사용자 가치**: 중간 (획득 채널)

## 5. 가격 경쟁력 분석

| 서비스 | 유료 시작가 | 원화 환산 (약) | vs Linkmap Pro |
|--------|------------|:-------------:|:--------------:|
| Infisical Pro | $18/identity/월 | ~₩25,000 | **2.5배 비쌈** |
| Doppler Team | $21/user/월 | ~₩29,000 | **2.9배 비쌈** |
| Port Standard | $40/seat/월 | ~₩55,000 | **5.6배 비쌈** |
| Cortex | ~$65/user/월 | ~₩90,000 | **9.1배 비쌈** |
| 1Password Dev | ~$20/월+ | ~₩28,000 | **2.8배 비쌈** |
| **Linkmap Pro** | **₩9,900/월** | **₩9,900** | **기준** |
| dotenvx | 무료 | ₩0 | 무료 (기능 적음) |
| Vault Community | 무료 | ₩0 | 무료 (복잡함) |

### 가격 전략 분석

**Linkmap의 가격 우위**:
- 글로벌 경쟁사 대비 **2.5~9배 저렴**
- 한국 시장 물가 감안 시 합리적 가격대 (월 만원 미만)
- 무료 티어로 진입 → Pro 전환 유도 경로 명확

**주의점**:
- 가격만으로는 Infisical/Doppler와 직접 비교 어려움 (기능 범위가 다름)
- Linkmap은 "더 적은 기능을 더 싸게"가 아닌 **"다른 가치를 적정 가격에"** 포지셔닝
- 무료 경쟁자(dotenvx, Vault Community)와는 **웹 UI + 시각화 + 마법사**로 차별화

## 6. 로드맵 제안

### 단기 (1~3개월) — "핵심 가치 강화"

| 우선순위 | 항목 | 출처 | 예상 효과 |
|:--------:|------|------|-----------|
| 1 | 환경변수 "되돌리기" 기능 | Infisical 벤치마킹 | 실수 복구로 초보자 안심 |
| 2 | DATABASE_URL 자동 조합 | Infisical 벤치마킹 | 가장 흔한 초보자 고충 해결 |
| 3 | 프로젝트 건강도 점수 | Cortex 벤치마킹 | 게이미피케이션으로 참여 유도 |
| 4 | "EnvKey 이전 가이드" 블로그 | EnvKey 종료 기회 | 이탈 사용자 획득 |

### 중기 (3~6개월) — "스마트 온보딩"

| 우선순위 | 항목 | 출처 | 예상 효과 |
|:--------:|------|------|-----------|
| 1 | package.json 기반 서비스 자동 감지 | Port 벤치마킹 | 온보딩 시간 90% 단축 |
| 2 | 프로젝트 템플릿 (Next.js + Supabase 등) | 자체 아이디어 | 원클릭 프로젝트 시작 |
| 3 | 서비스 카탈로그 50개 확장 | 자체 목표 | 커버리지 확대 |
| 4 | 영어 UI 완성 | 글로벌 확장 준비 | 해외 시장 진출 기반 |

### 장기 (6~12개월) — "플랫폼 확장"

| 우선순위 | 항목 | 출처 | 예상 효과 |
|:--------:|------|------|-----------|
| 1 | 시크릿 스캐닝 (Git 커밋 감지) | Infisical 벤치마킹 | 보안 사고 예방 |
| 2 | 노코드 서비스 연결 (OAuth 플로우) | Port 벤치마킹 | 코드 없이 서비스 연결 |
| 3 | 변경 알림 (슬랙/디스코드 연동) | Infisical 벤치마킹 | 팀 협업 강화 |
| 4 | 프로젝트 공유/포크 마켓플레이스 | 자체 아이디어 | 커뮤니티 성장 |

## 핵심 메시지

> **Linkmap은 "초보자의 첫 번째 환경변수 관리 도구"를 목표로 한다.**
>
> Infisical이 "시크릿 관리의 GitHub"이라면,
> Linkmap은 **"환경변수 관리의 Notion"** — 누구나 쉽게, 시각적으로, 바로 시작할 수 있는 도구다.
