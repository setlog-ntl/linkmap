|--------|-----------|-----------|-----------|------|:------------:|:-----------:|:---------:|

| **Configu** | Configuration-as-Code | 무료 영구 플랜 | 커스텀 | DevOps/CI-CD 팀 | ★★★☆☆ | 수일~수주 | 중간~높음 |

### 서비스별 상세 분석

#### Doppler — "가장 쉬운 시크릿 관리"
- **장점**: 직관적 UI, 자동 동기화, Git 스타일 협업, 빠른 온보딩
- **단점**: 최근 무료 티어 축소, 고급 기능 설정 복잡
- **vs Linkmap**: 순수 시크릿 관리에 집중. 서비스 맵·카탈로그·헬스체크 없음. 가격이 2배 이상 비쌈

#### HashiCorp Vault — "업계 표준이지만 진입장벽 높음"
- **장점**: 가장 포괄적 기능 (동적 시크릿, 암호화 서비스, PKI), 대규모 검증
- **단점**: 학습 곡선 가파름, 명령어 비일관성, 운영 복잡도 높음
- **vs Linkmap**: 완전히 다른 시장. Vault는 인프라 전문가용, Linkmap은 초보자용. 직접 경쟁 없음

#### dotenvx — "가장 단순한 대안"
- **장점**: 완전 무료, .env 워크플로 유지, ECIES(AES-256) 암호화, 최소 학습 곡선
- **단점**: CLI만 존재 (웹 UI 없음), 고급 기능 없음, 팀 관리 미약
- **vs Linkmap**: 가장 가까운 "간결함" 경쟁자. 그러나 UI·시각화·카탈로그·협업 기능 부재. Linkmap은 dotenvx의 "웹 UI + 시각화 상위호환"으로 포지셔닝 가능

#### EnvKey — "서비스 종료 (2025.02.01)"
- **상태**: 2024년 8월 사업 해체 발표, 2025년 2월 클라우드 종료
- **원인**: 재정적 지속가능성 문제 (인수 아님)
- **잔여**: GitHub에 오픈소스 도구·CLI 남아있으나 유지보수 없음
- **vs Linkmap**: **이탈 사용자 흡수 기회**. EnvKey와 유사한 "간편한 환경변수 관리" 가치를 제공하면서 더 많은 기능 보유

#### 1Password Developer — "기존 사용자 자연 확장"
- **장점**: 1Password 사용자에게 익숙한 인터페이스, IDE 통합, 검증된 보안
- **단점**: 무료 티어 없음, Vault Access Credit 모델 복잡, 독립 도구로는 매력 부족
- **vs Linkmap**: 1Password 생태계에 종속. 독립 시크릿 관리 도구로서의 매력은 Linkmap이 우위

#### Configu — "Configuration-as-Code"
- **장점**: 타입 시스템 기반 설정 검증, 정책 프레임워크, 멀티환경 동기화
- **단점**: 패러다임 전환 필요 (Configuration-as-Code), 초보자에게 어려움
- **vs Linkmap**: Configu는 "설정을 코드로" 접근법. Linkmap은 "설정을 시각적으로" 접근법. 철학이 다르지만 Configu의 타입 검증은 참고할 만함

## 2. 서비스 연결 시각화 / 개발자 포털 분야 (3개 서비스)

| 서비스 | 핵심 가치 | 무료 티어 | 유료 가격 | 타깃 | 초보자 친화도 | 온보딩 시간 | UI 복잡도 |
|--------|-----------|-----------|-----------|------|:------------:|:-----------:|:---------:|
| **Backstage** | 오픈소스 개발자 포털 (Spotify) | 완전 무료 (OSS) | Spotify Portal: 커스텀 | 100+ 엔지니어 조직 | ★☆☆☆☆ | 6~12개월 | 매우 높음 |
| **Port** | 노코드 개발자 포털 | 15석 무료 | $40/seat/월 | 전 규모 조직 | ★★★★★ | 수분 | 낮음 |
| **Cortex** | AI 기반 개발자 포털 | 없음 | ~$65-69/user/월 | 중대형 기업 | ★★☆☆☆ | 6개월+ | 높음 |

### 서비스별 상세 분석

#### Backstage — "강력하지만 거대한 투자 필요"
- **장점**: 완전 오픈소스, 최대 커뮤니티, CNCF 프로젝트, 50+ 플러그인, 무한 커스터마이징
- **단점**: 2-5명 전담 엔지니어 수년간 필요, 자체 빌드 필수, "배포"와 "실제 사용" 격차 큼
- **vs Linkmap**: 완전히 다른 스케일. Backstage는 수백명 조직의 플랫폼 엔지니어링 도구. Linkmap의 서비스 맵은 Backstage의 소프트웨어 카탈로그를 "초보자 버전"으로 재해석한 것

#### Port — "가장 접근성 높은 개발자 포털"
- **장점**: 노코드, 빠른 설정(수분), AI 에이전트 기능, 자동 서비스 발견, 15석 무료
- **단점**: $40/seat 가격 부담, SaaS 종속, 시크릿 관리 기능 없음
- **vs Linkmap**: Port의 **노코드 서비스 연결**과 **자동 발견** 컨셉은 벤치마킹 대상. 단, Port는 환경변수/시크릿 관리 없음 → Linkmap이 "환경변수 + 서비스 시각화" 결합으로 차별화

#### Cortex — "엔지니어링 성숙도 측정"
- **장점**: AI 기반 자동 서비스 발견·매핑, 스코어카드 (운영 준비도, DORA 메트릭), 50+ 통합
- **단점**: 무료 없음, ~$65/user 고가, 설정 경직 (UI-only 또는 YAML-only), 마찰로 인한 낮은 채택률
- **vs Linkmap**: Cortex의 **스코어카드** 컨셉은 벤치마킹 대상. "프로젝트 건강도 점수"로 단순화하여 적용 가능

### 핵심 발견: Linkmap만의 독점 영역

위 매트릭스에서 **Linkmap만 ✅인 기능** (시장 공백):

1. **서비스 맵 + 환경변수 관리 결합** — 어떤 경쟁사도 이 조합을 제공하지 않음
2. **설정 마법사** — 단계별 가이드로 초보자를 안내하는 서비스 없음
3. **.env 파일 임포트 웹 UI** — CLI 없이 드래그&드롭으로 가능한 서비스 없음
4. **서비스별 헬스 체크 + 환경변수 관리 결합** — 연결 상태 확인까지 통합한 서비스 없음
5. **MCP 서버 네이티브 통합** — AI 코딩 도구와의 직접 연동은 시장에 없음
6. **Korean-first + 초보자 타깃** — 한국어 우선 환경변수 관리 도구는 시장에 전무
