## 핵심 인사이트 3줄 요약

1. **Linkmap만의 독점 영역**: "환경변수 관리 + 서비스 맵 시각화"를 결합한 서비스는 10개 경쟁사 중 **Linkmap이 유일**하다. 설정 마법사, .env 임포트 UI, 헬스 체크 통합도 시장에 없는 기능이다.

2. **초보자 시장은 비어 있다**: 기존 시크릿 관리 도구(Infisical, Vault, Doppler)는 모두 DevOps/숙련 개발자를 타깃으로 한다. "바이브코딩 초보자"를 위한 환경변수 관리 도구는 **시장에 전무**하며, 이것이 Linkmap의 핵심 기회다.

3. **가격 경쟁력 확보**: Linkmap Pro(₩9,900/월)는 Infisical Pro($18)의 40%, Doppler Team($21)의 34% 수준이다. 단, 가격이 아닌 **"다른 가치(시각화·간결함·한국어)"를 적정 가격에** 제공하는 것이 올바른 포지셔닝이다.

- **소스**: 각 서비스 공식 사이트, 가격 페이지, 문서, G2/PeerSpot 리뷰, GitHub
